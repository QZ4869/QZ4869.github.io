软件推荐（win10）

由于换过以及重装过多次电脑（ThinkPad质量越来越差了），被迫整理过一个“装机必备”列表及一些相应的portable版软件压缩包和安装包，用来一次性装好常用的不错的软件，其中大部分都是绿色开源的（不特别说明就是），在这里做一次推荐。由于我主要用win10，所以这是针对win10的，但实际上大部分其实都是支持多平台的。

我的装机必备列表

Google Chrome 浏览器

Chrome 浏览器在国内并不方便下载和安装，可从下面链接下载离线安装包ChromeStandaloneSetup64.exe，并最好在不联网情况下（或者教育网得连ipv6）安装

  • 下载链接：https://www.google.cn/intl/zh-CN/chrome/thank-you.html?installdataindex=empty&standalone=1&statcb=0
  • 可参考：https://www.zhihu.com/question/27832629?sort=created

Everything

本地文件搜索工具，用于本地文件搜索，速度特别快，因为原理是读取NTFS文件系统的日志（USN Journal）。所以最大缺点也就是只支持NTFS系统。

  • 最好下载官网最新版（绿色便携版），之前的版本我用正则表达式搜索时发现了bug，然后下了最新版发现已经解决了
  • 官网：https://www.voidtools.com/zh-cn/downloads/
  • 注意不了解的话不要启用Everything的服务器功能，可能会让别人能直接通过Everything访问你的本地文件（之前看到过新闻说Everything的HTTP服务器有漏洞）
  • 建议设为开机启动

BANDIZIP

支持绝大多数格式的解压缩软件，虽然有便携版（portable）但更推荐安装版。

  • 官网：http://www.bandisoft.com/

Snipaste

非常方便的截图软件，功能也很多，详见官网

  • 官网：https://www.snipaste.com/
  • 建议设为开机启动，F1直接截图

Python3

这是必备的，非常非常强大，即使对于不会自己写python代码的人来说也推荐安装，因为网上有非常多好用的库可以直接安装，然后照着库提供的使用文档就能直接用。之后应该会专门写一篇来推荐python的各种库

  • 官网：https://www.python.org/
  • 选择一个版本下载它的安装器Windows x86-64 executable installer，之后按提示一步步安装就行，比如我现在用的就是python-3.7.4-amd64.exe，并一直保留着，因为可以用这个安装器方便的卸载和重装

SumatraPDF

支持绝大多数格式的开源轻量级阅读器

  • 对pdf文件的支持要弱于Adobe Reader，特别是一些新特性比如动图的支持
  • 主要是用于epub和djvu格式的文件，pdf文件我还是用Adobe Reader
  • 由于来自知网的一些文献为caj格式，电脑还是得备上知网提供的专门阅读器cajviewer，不过据说知网现在提供所有文献的pdf版本
  • 官网：https://www.sumatrapdfreader.org/free-pdf-reader.html

Adobe Reader

Adobe Reader还是得装，其他pdf阅读器一般对pdf的支持都有细微的问题

  • 非绿色便携版

calibre

也许是最强大的开源电子书软件

  • 阅读器就用calibre+Adobe Reader+SumatraPDF这三个就行了，而calibre是其中最强大的
  • 官网：https://calibre-ebook.com/download

Sublime Text

非常轻便强大的编辑器

  • 官网：http://www.sublimetext.com/
  • 由于启动速度非常快，所以我一般对简单的文本编辑都用sublime，包括python和ROOT，而对于较复杂的包括涉及编译的，就用VScode
  • 缺点：经常会弹出Purchase信息（似乎是根据Ctrl+s次数来弹出）

Visual Studio Code

非常强大的代码编辑器，以前我是windows用sublime而Linux用Atom，但自从微软出了VScode后我就没装过Atom了，一般写Markdown、Latex都用VScode

  • 推荐扩展插件（Extensions）：
    • Chinese (Simplified) Language Pack
    • Markdown Preview Enhanced
    • markdown-index
    • 翻译(英汉词典)
    • LaTeX Workshop
    • Remote-SSH
  • 官网：https://code.visualstudio.com/

GoldenDict

开源词典，非常强大，主要是可以自己放相应的字典

  • Github上的下载页面：https://github.com/goldendict/goldendict/wiki/Early-Access-Builds-for-Windows
  • 一个免费字典网站

PotPlayer 64 bit

也许是最好用最强大的播放器，但要记得装一个解码器，不然有些视频会没声音

  • OpenCodec解码器下载：https://daumpotplayer.com/download/ 的 OpenCodecSetup64.exe
  • 官网：https://daumpotplayer.com/download/

utorrent

常用的torrent下载客户端，我专门用于北邮人

Motrix

一款全能的开源下载工具

  • 比浏览器自带下载更快，更全面
  • 官网：https://motrix.app/zh-CN/

TeX Live 2019

用于LaTeX编译，适用于所有平台，同一个安装包，一般在tuna镜像站下载

  • 这是现在最常用的LaTeX发行版，现在用CTeX的人应该已经很少了，之后可能发一篇win10下使用LaTeX的文档
  • tuna镜像：https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/

CERN ROOT

CERN开发的数据分析软件，基于C++，一般用在Linux上，但提供的windows版也很方便，我在windows上写简单的C++程序时都直接在ROOT上运行，而无需装Visual Studio这种巨无霸

  • 官网：https://root.cern.ch/
    • windows使用ROOT5更方便，即下载root_v5.34.38.win32.vc12

putty

SSH和Telnet客户端，一般用于SSH连接服务器，很轻便，用的人很多，但我自从下了MobaXterm就再没用过了

  • 官网：https://www.chiark.greenend.org.uk/~sgtatham/putty/

MobaXterm

可以说是一个全能的用于使用Linux服务器的工具，支持SSH、FTP等各种网络工具，支持X Server，还自带一个终端，正如官网的说法：

MobaXterm is your ultimate toolbox for remote computing.
…
MobaXterm provides all the important remote network tools (SSH, X11, RDP, VNC, FTP, MOSH, …) and Unix commands (bash, ls, cat, sed, grep, awk, rsync, …) to Windows desktop, in a single portable exe file which works out of the box.
There are many advantages of having an All-In-One network application for your remote tasks, e.g. when you use SSH to connect to a remote server, a graphical SFTP browser will automatically pop up in order to directly edit your remote files. Your remote applications will also display seamlessly on your Windows desktop using the embedded X server.

  • 官网：https://mobaxterm.mobatek.net/ 同时提供付费版，但是免费版完全够了

mathpix OCR

用于数学公式截图识别转化为LaTeX代码

  • 设为开机启动，快捷键设为Ctrl+Alt+m，非常方便，但需要联网，免费用户有1000次/月的次数限制
  • 非绿色便携版
  • 官网：https://mathpix.com/ocr

Mathematica

强大的数学物理软件（也很庞大），特别是符号运算，应该是我现在用的唯一一个破解版软件，北邮人上下的。以前用得稍微多一些，但现在有时候做一些快速数值计算和公式符号运算时还是用Mathematica，最方便

Git

主要用于文本文件的版本管理，要专门学习下才会用，但对于完全不需要掌握git的人来说，只是它提供的git-bash就非常有用了，因为git-bash使我们能够在windows下运行很多Linux命令

  • 官网：https://git-scm.com/

GeoGebra

开源几何作图软件，支持二维、三维图和函数作图，以及动画

  • 高中时候习惯了用几何画板（Sketchpad5），所以用GeoGebra一直有些不适应，但不得不承认GeoGebra要优秀得多
  • 官网：https://www.geogebra.org/

Blender

非常强大的开源跨平台全能三维动画制作软件，只有约300M，但是完全可以取代各种收费的专业的3D建模、视频编辑等其他相关软件，唯一的缺点可能是门槛稍高

  • 官网：https://www.blender.org/

irreader

用于RSS订阅，我才用，因为我需要测试下这个网站的RSS功能，下载irreader后，添加我这个网站的RSS源（也就是我网站页面的左边栏的左下角那个标志）:https://qz4869.github.io/atom.xml，就可以及时收到网站更新的文章。

  • 很多邮箱客户端直接有RSS订阅功能，所以完全不需要irreader，但我用的win10自带的邮箱客户端，并没有RSS订阅功能。对比了很多RSS客户端，这个不仅开源免费的而且功能最多，唯一的缺点是界面丑了一点
  • 官网：http://irreader.fatecore.com/
  • 源码：https://github.com/skdjfla/irreader

其他软件

  • 搜狗输入法
    虽然不喜欢，但是以前试过很多输入法还是搜狗输入法比较用起来比较顺手，一开始设置没注意会有些弹窗和小bug，但还是能解决的。我一般将它隐藏在语言栏
  • 网易云音乐
    我的装机必备
  • miMind
    开源思维导图软件，现在稍微好用的思维导图软件一般都要收费，但这个好用且开源免费，只是中文字体需要自己折腾下
  • ScreenToGif
    开源录屏转gif，很好用
  • captura
    非常强大的开源录屏软件，很好用。其中音频的输入来源的设置有些讲究，录之前先试试每个选项及它们组合出来的效果（比如如果要录会议，就既要有麦克风来源也要有耳机来源，不然听不到自己的声音）
  • krita
    开源画画软件
  • Steam
    游戏商店
  • Celestia
    开源3D天文软件
  • Lenovo联想驱动管理
    这似乎是当时新电脑随机附送的软件，用起来还可以不碍事
  • Pulse Secure
    外网连学校SSLVPN系统的推荐客户端，Linux下用OpenConnect
  • ZOOM
    这段时间试了六个会议、上课软件，功能上比较突出的就是ZOOM，白板功能很方便，录屏据说也很小，另外推荐huiyizhuo，虽然没用过但官网给我的感觉很不错

装机其他事项

  • 在系统的事件查看器查看开机次数，以及使用CrystalDiskInfo和CrystalDiskMark查看硬盘情况（电脑使用一段时间后也可以检测检测）
    • 官网：https://crystalmark.info/en/software/
  • 开机密码、指纹
  • 电源设置（盖板，电源键）
  • 使开始菜单中电源处右键出现休眠选项（电源->更改不可用设置）
  • 自带邮箱客户端的邮箱登录
  • Chrome的一系列密码
  • 无线网Tsinghua-Secure的密码
  • 双系统Ubuntu
    这是以前装机的最后一步，不过现在有了服务器后就懒得装双系统了
